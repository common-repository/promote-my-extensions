== Description ==

This plugin creates a custom post type to share information about WordPress themes and plugins you've developed.

= Extra Details =

* Allows you to create sub pages and indicate which ones has the plugin documentation.
* Provide a URL for where a person can download the plugin.
* Add a quick link to some Frequently Asked Questions.
* Add a quick link to the support forums.
* Add a quick link to the plugin reviews.
* Add a link to a donate button to allow people to donate to your plugin.

== Frequently Asked Questions ==

= What does this plugin actually do? =

It creates custom post type allowing you to share information about your WordPress themes and plugins.
